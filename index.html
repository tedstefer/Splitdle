How It Works

1. Start

Install our browser extension, click the Start Daily Run button, and get sent through four daily "-dle" games.

2. Solve

Race through each games back to back. Your splits are tracked automatically. Everyone plays the same four games every day.

3. Share

Finish your run and get a shareable result card showing your splits and final time. Share and compare with friends!

Extension Required

To start a Splitdle run, you need the browser extension installed.
